sinatra-template
================

A Sinatra template.

Features: Bootstrap, Compass, Slim, CoffeeScript, Sass, Asset pipeline.

Install dependencies
-----

    git clone https://github.com/christianhellsten/sinatra-template.git
    cd sinatra-template
    bower install
    bundle

Start server
-----

    puma
    puma -p 3000
    puma -e production
